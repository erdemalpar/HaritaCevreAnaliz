# CBS Analiz Sistemi 🌍

Bu proje, **OpenStreetMap (Overpass API)** verilerini kullanarak belirli bir konumun (enlem/boylam) çevresindeki önemli noktaları (Eğitim, Sağlık, Kamu, Ulaşım, Alışveriş vb.) analiz eden ve görselleştiren gelişmiş bir **Coğrafi Bilgi Sistemi (CBS)** aracıdır.

![CBS Analiz Sistemi](https://github.com/user-attachments/assets/demo_placeholder.png)

## 🚀 Özellikler

- **Dinamik Harita:** Leaflet.js altyapısı ile etkileşimli harita deneyimi.
- **Detaylı Analiz:** Overpass API ile gerçek zamanlı veri çekme.
- **Kategori Filtreleme:** Eğitim, Sağlık, Kamu, Tapu/Kadastro ve daha fazlası.
- **Mesafe Ölçümü:** Merkeze olan mesafelerin otomatik hesaplanması ve görselleştirilmesi.
- **TKGM Entegrasyonu:** Parsel sorgulama için doğrudan TKGM bağlantıları.
- **Harita Katmanları:** Standart (OSM), Uydu ve Hibrit görünüm seçenekleri.

## 🛠️ Teknolojiler

- **React:** Kullanıcı arayüzü mantığı.
- **Tailwind CSS:** Modern ve premium tasarım.
- **Leaflet.js:** Harita motoru.
- **Lucide React:** İkon seti.
- **Vite:** Hızlı geliştirme ortamı.

## 📦 Kurulum

Projeyi yerelinizde çalıştırmak için aşağıdaki adımları izleyin:

1.  Depoyu klonlayın:
    ```bash
    git clone https://github.com/erdemalpar/HaritaCevreAnaliz.git
    ```
2.  Proje dizinine gidin:
    ```bash
    cd HaritaCevreAnaliz
    ```
3.  Bağımlılıkları yükleyin:
    ```bash
    npm install
    ```
4.  Geliştirme sunucusunu başlatın:
    ```bash
    npm run dev
    ```
